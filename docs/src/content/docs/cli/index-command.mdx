---
title: lucerna index
description: One-shot indexing of a project directory.
---

Index an entire project directory in one pass. Defaults to the current working directory.

```bash
lucerna index
lucerna index --dir /path/to/project
```

If no `lucerna.config.ts` is found, one is created automatically in the current directory.

## Options

| Option | Description |
|--------|-------------|
| `--dir <path>` | Project root directory (default: current working directory) |
| `--storage-dir <dir>` | Override storage directory (default: `<root>/.lucerna`) |
| `--include <globs>` | Comma-separated include glob patterns (overrides config file) |
| `--exclude <globs>` | Additional comma-separated exclude glob patterns (overrides config file) |
| `--no-semantic` | Disable semantic search (BM25 only) |

## Example output

```
Indexing /path/to/project...
Done. 142 files, 3817 chunks indexed.
```

:::tip
Place a `lucerna.config.ts` at the project root (or any parent directory) to configure embeddings and reranking without passing flags every time. See [Configuration](/configuration/config-file).
:::
