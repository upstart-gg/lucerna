---
title: lucerna search
description: Search an existing index using hybrid semantic + BM25 search.
---

Search an indexed project with natural language or symbol names.

```bash
lucerna search /path/to/project "authentication middleware"
```

## Options

| Option | Description |
|--------|-------------|
| `--no-semantic` | Disable semantic search (BM25 only) |
| `--embedder <provider:model>` | Embedding provider and model, e.g. `voyage:voyage-4`. Providers: openai, cohere, voyage, jina, cloudflare, mistral, gemini, vertex, ollama |
| `--reranker <provider:model>` | Reranker provider and model, e.g. `voyage:rerank-2.5`. Providers: voyage, cohere, jina, cloudflare |
| `--config <path>` | Path to `lucerna.config.ts` / `lucerna.config.js` |
| `--limit <n>` | Max results (default: `10`) |
| `--format raw\|json\|pretty-json` | Output format (default: `raw`) |
| `--language <lang>` | Filter by language: `typescript`, `javascript`, `json`, `markdown` |
| `--type <type>` | Filter by chunk type: `function`, `class`, `method`, `interface`, … |

### Compact JSON (`--format json`)

Ideal for piping:

```bash
lucerna search /path/to/project "verifyToken" --format json | jq '.[0].id'
```
