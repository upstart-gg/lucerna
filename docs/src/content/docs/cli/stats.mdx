---
title: lucerna stats
description: Show index statistics for a project.
---

Display statistics about an indexed project. Defaults to the current working directory.

```bash
lucerna stats
lucerna stats --dir /path/to/project
```

## Options

| Option | Description |
|--------|-------------|
| `--dir <path>` | Project root directory (default: current working directory) |
| `--format raw\|json\|pretty-json` | Output format (default: `raw`) |

## Raw output

```
Project:        /path/to/project
Project ID:     a1b2c3d4e5f6
Total files:    142
Total chunks:   3817
Last indexed:   2025-04-15T10:00:00.000Z
```
