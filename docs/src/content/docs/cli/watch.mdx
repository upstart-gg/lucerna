---
title: lucerna watch
description: Index a project and re-index on file changes.
---

Index a project and automatically re-index files as they change. `.gitignore` files at any depth are respected automatically. Defaults to the current working directory.

```bash
lucerna watch
lucerna watch --dir /path/to/project
```

## Options

All options from [`index`](/cli/index-command) (`--dir`, `--storage-dir`, `--include`, `--exclude`, `--no-semantic`), plus:

| Option | Description |
|--------|-------------|
| `--debounce <ms>` | File-change debounce delay (default: `500`) |

## Example output

```
Indexing /path/to/project...
Initial index complete. 142 files, 3817 chunks.
Watching for changes. Press Ctrl+C to stop.
[indexed] src/auth/middleware.ts (12 chunks)
[indexed] src/db/pool.ts (8 chunks)
[removed] src/legacy/old-auth.ts
[error]   src/broken.ts: Unexpected token at line 42
```
