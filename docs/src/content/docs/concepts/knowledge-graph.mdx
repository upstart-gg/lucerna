---
title: Knowledge Graph
description: Symbol relationship graph extracted from the AST.
---

Lucerna extracts a directed relationship graph from the AST alongside chunks. Edges are stored in a LanceDB table and can be traversed with the [`lucerna graph`](/cli/graph) CLI command.

## Relationship types

| Type | Meaning |
|---|---|
| `CALLS` | A function/method calls another function/method |
| `IMPORTS` | A file's import block references another module |
| `DEFINES` | The import chunk maps to each named declaration in the file |
| `EXTENDS` | A class extends another class |
| `IMPLEMENTS` | A class implements an interface |
| `USES` | A chunk references a type or variable defined elsewhere |

## Edge IDs

Every edge has a stable ID computed from `projectId + sourceChunkId + targetChunkId + type`. Duplicate edges are upserted rather than duplicated.
