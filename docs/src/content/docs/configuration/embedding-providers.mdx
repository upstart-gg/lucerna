---
title: Embedding Providers
description: Embedding providers for semantic (vector) search — recommended models and pricing.
---

Lucerna supports multiple embedding providers for semantic (vector) search. The recommended way to configure them is via your `.mcp.json` file — see [Env Vars](/configuration/env-vars) for the full variable reference, or [Custom Providers](/configuration/custom-providers) for the `lucerna.config.ts` approach.

:::note[Pricing]
Prices shown are approximate as of April 2026 and subject to change. Always verify at the provider's pricing page before budgeting.
:::

---

Each snippet below shows the `env` block to add to your `.mcp.json`:

```json title=".mcp.json"
{
  "mcpServers": {
    "lucerna": {
      "command": "npx",
      "args": ["-y", "@upstart.gg/lucerna", "mcp-server"],
      "env": {
        "LUCERNA_EMBEDDING": "voyage:voyage-4",
        "VOYAGE_API_KEY": "your-key"
      }
    }
  }
}
```

:::tip[Using unlisted models]
For models not in the tables below, pass dimensions as a third segment:

```json
{ "LUCERNA_EMBEDDING": "ollama:my-custom-model:768" }
```

OpenAI's `text-embedding-3` models also support server-side Matryoshka truncation — passing a smaller dimension produces shorter vectors directly from the API (better quality than post-hoc truncation):

```json
{ "LUCERNA_EMBEDDING": "openai:text-embedding-3-large:256" }
```
:::

### Voyage ⭐ recommended for code

Best-in-class for code retrieval. `voyage-4` is the latest generation model recommended for code search.

```json title=".mcp.json → env"
{
  "LUCERNA_EMBEDDING": "voyage:voyage-4",
  "VOYAGE_API_KEY": "your-key"
}
```

| Model | Dimensions | Price per 1M tokens | Notes |
|-------|-----------|---------------------|-------|
| `voyage-4` | 1024 | — | Latest generation — **top pick** |
| `voyage-4-lite` | 512 | — | Faster, lower cost |
| `voyage-code-3` | 1024 | $0.18 | Legacy, code-optimized, 32K ctx |
| `voyage-3` | 1024 | $0.06 | General text |
| `voyage-3-lite` | 512 | $0.02 | Faster, lower cost |

### GCP Vertex AI

Requires a GCP project and an access token. `GOOGLE_CLOUD_LOCATION` is optional (default: `us-central1`).

```json title=".mcp.json → env"
{
  "LUCERNA_EMBEDDING": "vertex:text-embedding-005",
  "GOOGLE_CLOUD_PROJECT": "my-gcp-project",
  "GOOGLE_ACCESS_TOKEN": "obtain via: gcloud auth print-access-token"
}
```

| Model | Dimensions | Price per 1M tokens | Notes |
|-------|-----------|---------------------|-------|
| `text-embedding-005` | 768 | $0.10 | Latest, recommended |
| `text-embedding-004` | 768 | $0.10 | Previous generation |
| `text-multilingual-embedding-002` | 768 | $0.10 | Multilingual |

### Cloudflare Workers AI

Free tier: 10K neurons/day (~9M tokens/day for bge-m3).

```json title=".mcp.json → env"
{
  "LUCERNA_EMBEDDING": "cloudflare:@cf/baai/bge-m3",
  "CLOUDFLARE_ACCOUNT_ID": "your-account-id",
  "CLOUDFLARE_API_TOKEN": "your-token"
}
```

| Model | Price per 1M tokens | Notes |
|-------|---------------------|-------|
| `@cf/baai/bge-m3` | ~$0.012 (free tier applies) | Multilingual, good baseline |

### Ollama (local GPU)

No API key required. Requires a running Ollama server. `OLLAMA_HOST` is optional (default: `http://localhost:11434`).

```json title=".mcp.json → env"
{
  "LUCERNA_EMBEDDING": "ollama:nomic-embed-text"
}
```

| Model | Dimensions | Price | Notes |
|-------|-----------|-------|-------|
| `nomic-embed-text` | 768 | Free | Popular, fast |
| `mxbai-embed-large` | 1024 | Free | Higher quality, GPU recommended |
