## Provider selection

| Variable | Format | Example |
|----------|--------|---------|
| `LUCERNA_EMBEDDING` | `provider:model` or `provider:model:dimensions` | `voyage:voyage-code-3`, `ollama:my-custom-model:768` |
| `LUCERNA_RERANKING` | `provider:model` | `cohere:rerank-english-v3.0` |

Both `provider` and `model` are always required. Using only a provider name is an error.

The optional third segment `dimensions` is a positive integer. Use it for models not listed in the built-in table (e.g. custom Ollama models, Cloudflare variants) or to enable OpenAI Matryoshka truncation (server-side dimension reduction):

```bash
LUCERNA_EMBEDDING=ollama:my-custom-model:768
LUCERNA_EMBEDDING=openai:text-embedding-3-large:256
```

If `LUCERNA_EMBEDDING` is not set, semantic search is disabled and only lexical (BM25) search is available. The MCP server will include a warning in tool results.

CLI flags (`--embedder`, `--reranker`) take priority over env vars.

## Provider credentials

| Variable | Provider |
|----------|---------|
| `OPENAI_API_KEY` | OpenAI |
| `COHERE_API_KEY` | Cohere (embeddings + reranking) |
| `VOYAGE_API_KEY` | Voyage (embeddings + reranking) |
| `JINA_API_KEY` | Jina (embeddings + reranking) |
| `MISTRAL_API_KEY` | Mistral |
| `GOOGLE_API_KEY` | Google Gemini |
| `GOOGLE_CLOUD_PROJECT` | GCP Vertex AI — project ID |
| `GOOGLE_CLOUD_LOCATION` | GCP Vertex AI — region (default: `us-central1`) |
| `GOOGLE_ACCESS_TOKEN` | GCP Vertex AI — OAuth access token (`gcloud auth print-access-token`) |
| `CLOUDFLARE_ACCOUNT_ID` | Cloudflare Workers AI |
| `CLOUDFLARE_API_TOKEN` | Cloudflare Workers AI |
| `OLLAMA_HOST` | Ollama (default: `http://localhost:11434`) |
| `LMSTUDIO_BASE_URL` | LM Studio (default: `http://localhost:1234`) |

## Other variables

| Variable | Default | Description |
|----------|---------|-------------|
| `LUCERNA_STORAGE` | `.lucerna` in project root | Override where Lucerna stores its index |

## Example: MCP server config

```json
{
  "mcpServers": {
    "lucerna": {
      "command": "npx",
      "args": ["-y", "lucerna", "mcp-server"],
      "env": {
        "LUCERNA_EMBEDDING": "voyage:voyage-code-3",
        "VOYAGE_API_KEY": "your-key"
      }
    }
  }
}
```
