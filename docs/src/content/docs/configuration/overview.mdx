---
title: Configuration
description: Configure embedding and reranking providers via lucerna.config.ts or env vars.
---

Lucerna requires an embedding provider to enable semantic (vector) search. Without one, only lexical (BM25) search is available.

The two main ways to configure providers are:

1. **Environment variables** — set `LUCERNA_EMBEDDING=provider:model` (picked up by the MCP server and CLI automatically)
2. **Config file** — place a `lucerna.config.ts` at the project root

## Config file

Supported filenames (checked in order):

- `lucerna.config.ts`
- `lucerna.config.js`
- `lucerna.config.mjs`

```ts
// lucerna.config.ts
import { VoyageEmbeddings, VoyageReranker } from '@upstart.gg/lucerna';

export default {
  embeddingFunction: new VoyageEmbeddings({ model: 'voyage-code-3' }),
  rerankingFunction: new VoyageReranker({ model: 'rerank-2' }),
};
```

The MCP server and all CLI commands pick up `lucerna.config.ts` automatically from the project root.

## Passing providers via CLI flags

Use `--embedder` and `--reranker` flags for a one-off run (both require `provider:model` format):

```bash
lucerna mcp-server . --embedder voyage:voyage-code-3 --reranker voyage:rerank-2
lucerna index . --config /path/to/lucerna.config.ts
```

## Priority order

| Priority | Source |
|----------|--------|
| 1 (highest) | `--no-semantic` flag (disables embeddings entirely) |
| 2 | `--embedder` / `--reranker` CLI flags |
| 3 | `lucerna.config.ts` / `.js` |
| 4 (lowest) | `LUCERNA_EMBEDDING` / `LUCERNA_RERANKING` env vars |

## Pointing to a custom config path

```bash
lucerna mcp-server . --config /path/to/my.config.ts
lucerna index . --config ./configs/lucerna.dev.ts
```

## Type reference

```ts
interface LucernaConfig {
  /**
   * Embedding provider instance, or `false` to disable semantic search (BM25 only).
   * Omit to fall back to the LUCERNA_EMBEDDING env var, or no semantic search if unset.
   */
  embeddingFunction?: EmbeddingFunction | false;

  /**
   * Reranking provider instance, or `false` to disable reranking.
   * Omit for no reranking (the default).
   */
  rerankingFunction?: RerankingFunction | false;
}
```
